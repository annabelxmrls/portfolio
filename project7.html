Media Player

November 2023

About This Project

This project is from my web design with JavaScript class in the fall semester of 2023. I was given the assignment to create a webpage containing a media player displaying captions, triggers on time stamps, and playback controls. I decided to take a music preformance from a Victoria's Secret Fashion Show to present wikipedia pages of the models as they walk down the runway. The music preformance by Rihanna adds an entertainment aspect as the wikipedia pages add an informational aspect.
