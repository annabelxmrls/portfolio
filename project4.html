Noah Franklin Homestead

May 2023

About This Project

This project is from my web design class in the spring semester of 2023. I was given the assignment to create a multi-page website for a ficitional bed and breakfast inn called Noah Franklin Homestead. I was provided with the content that includes images, room details, special deals, and contact information and I had to plan out how I would present each piece of content through wireframes and style tiles.

Skills Used

  • User Experience/Interaction Design
  • Wireframing
  • Style Tiles
  • CSS
  • Web Design/Development
