About This Project

This project is from my user experience design class in the fall semester of 2023. I was given the assignment to pitch an app idea with the purpose of helping a community. I chose to create an app that will cater towards students and create a planner app for their assignments. This project includes a pitch deck and a prototype that was all created in Figma.
