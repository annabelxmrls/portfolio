Culinary Roulette

December 2023

About This Project

This project is from my web design class in the fall semester of 2023. I was given the assignment to create a custom web app that collects remote data and meaningfully processes it for a targeted audience. I thought of a fun way to use a recipe API and create a recipe generator that gives the user a recipe based on the category they choose.
